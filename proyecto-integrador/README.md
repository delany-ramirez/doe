# Proyecto integrador

> ✅ **Completo** (Fase 9): enunciado, datos y rúbrica.

Proyecto final que integra lo aprendido en las cuatro semanas: planeación de un
experimento, análisis factorial con ANOVA y verificación de supuestos, interpretación de
interacciones, diagnóstico de curvatura y la decisión de pasar a superficie de respuesta
(RSM).

## Contenido

| Archivo | Descripción |
|---------|-------------|
| [`enunciado.md`](enunciado.md) | Descripción del problema, entregables y cronograma |
| [`rubrica.md`](rubrica.md) | Criterios de evaluación (100 pts) y resultados esperados |
| [`datos/extraccion.csv`](datos/extraccion.csv) | $2^3$ con 2 réplicas + 4 puntos centrales; rendimiento de extracción |

## Caso

Optimización del **rendimiento de extracción** de un compuesto activo en función de
**temperatura**, **tiempo** y **pH**. El conjunto de datos exhibe efectos principales,
interacciones y **curvatura** (puntos centrales), de modo que el proyecto recorre el ciclo
completo: factorial → diagnóstico → siguiente paso hacia RSM.

> Reutiliza como plantilla los notebooks de las semanas 2–4.
