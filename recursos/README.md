# Recursos generales

## Contenido

| Recurso | Descripción |
|---------|-------------|
| [`bibliografia.md`](bibliografia.md) | Texto principal (Montgomery), complementarios y artículos clásicos |

## Cómo usar estos recursos

- Consulta el **cheatsheet** al elegir un diseño o al interpretar una tabla ANOVA.
- Usa la **plantilla de informe** y el **checklist** en cada ejercicio y en el proyecto
  integrador.
- La **bibliografía** y los **enlaces** son para profundizar más allá del temario.
